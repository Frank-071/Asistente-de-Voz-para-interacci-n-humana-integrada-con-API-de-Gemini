# Asistente-de-Voz-para-interacci-n-humana-integrada-con-API-de-Gemini
Un prototipo de asistente que puede reconocer su nombre "Pandora" (configurable) y contestar a preguntas un tanto básicas, puede ajustar sus respuestas en base a una previa configuración. Está conectada con una API de Google (Gemini) para que sus respuestas sean más precisas.
A continuación anexo el manual para (entre otras cosas) poder entrenar y ejecutar el modelo: https://drive.google.com/file/d/1JZv1_JIho3sbFd2r5zFGfu0RCreiJmii/view?usp=drive_link
